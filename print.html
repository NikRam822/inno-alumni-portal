Project Description

The inno-alumni-portal serves as a dedicated website aimed at facilitating various services for the alumni community of Innopolis University. It offers alumni the opportunity to stay updated with university events, register for elective courses, manage attendance, and request passes for accessing university facilities.

  • Customer Complaints

One of the notable features of the system is its seamless integration for customer feedback and complaint submission. Users can directly submit their complaints through the designated Telegram group mentioned on the website.

  • Users Data Management

Administrators have the authority to manage user data effectively. They can review and process requests from users, providing feedback or taking necessary actions such as acceptance or denial based on specific reasons.

  • Pass Order

Alumni users have the convenience of requesting passes to access university facilities for various purposes directly from the main page of the website. This streamlined process enhances accessibility and convenience for the alumni community.

  • Elective Courses Participation Requests

Another significant aspect of the inno-alumni-portal is its functionality allowing alumni users to request participation in elective courses offered by the university. Through this feature, alumni can express their interest and attend courses relevant to their preferences and schedules.

Project "IU Alumni" Overview

The IU Alumni project encompasses a wide range of functionalities geared towards managing university events, facilitating elective course registration, tracking attendance, and processing pass requests for accessing university premises. It is designed to cater to the diverse needs and preferences of the university alumni community.

Conclusion

In summary, the inno-alumni-portal serves as a centralized platform aimed at enhancing the alumni experience at Innopolis University. With its array of features and user-friendly interface, it strives to foster engagement, facilitate communication, and streamline processes for the benefit of all alumni stakeholders.

This documentation aims to provide comprehensive insights into the functionality and objectives of the inno-alumni-portal, empowering users to leverage its capabilities effectively.

For users

As a user of the inno-alumni-portal, you have access to a range of features tailored to enhance your experience and connectivity with Innopolis University. From registering for elective courses to staying updated with university events, managing attendance, and submitting feedback, this guide will walk you through the process step-by-step.

Section overview

The system has pages such as:

  • Users login page
  • Overview
  • Request Pass
  • Request Electives
  • Make Donations
  • Account

Detailed information on each section is provided below.

Users login page

Authorisation and Authentication page. For authentication and authorisation, you can use several options:

  • Register.
  • Sign in if you already have an account.
  • Sign in using the Innopolis University authentication system if you have a corporate account.

Overview

After successfully logging in to your account, you will be taken to the Overview section. In this section on the left side there is a menu to navigate between the sections. Also on this page are brief descriptions of the sections.

Request Pass

This page is designed to generate and send enquiries. You can fill in the necessary forms and send a request if you have a need. In the right part of the page there is background information on popular issues.

Once you have completed the form, your enquiry will appear in the History section. Here you can track the status of your enquiry and find out more details.

For even more detailed information you can click on an enquiry and see an enquiry card with detailed information about the enquiry.

Request Electives

This page is designed to generate and send requests for elective courses. You can select the available electives and submit a course request.

You can also track the history of requests at the bottom of the section and track the status of requests. Also, if you click on a request, you can read detailed information.

Make Donations

This section provides information about the possibility of donations. The main part of the section provides detailed information and motivation for action.

At the bottom of the section are the necessary details and donation forms.

Account

This section provides detailed information about the account you are using. You can also update the account data here.

For administrators

As an administrator responsible for overseeing the operations and functionalities of the inno-alumni-portal, this guide provides valuable insights into managing user data, processing pass requests, overseeing elective course registrations, and addressing user feedback. You'll find detailed instructions and best practices to streamline administrative tasks and ensure a seamless experience for all users.

Section overview

  • Overview

Detailed information on each section is provided below.

Admins login page

System login page. You need to have an admin account to log in to authenticate and authorise.

Overview

After authentication and authorisation, you are taken to the main page. Here you can see the data of the last logged in users.

Pass Requests

This page contains requests from users. Here you can see the dates and statuses of the requests.

You can click on the required enquiry for more information and to complete the enquiry response form.

Manage Elective Courses

This section is similar to the previous one, but belongs to the elective courses section. Here you can familiarise yourself with user requests.

To respond to a request, you need to fill in the response form, which can be opened by clicking on the request.

Manage Donation

In this section we can update the information about donations, as well as review the donations that have come in.

By clicking on a specific donation, we can get more detailed information about the selected donation.

Account

This section provides information about the users of the system. You can also add new users with the required access by clicking the add button in the upper right corner of the screen.

How to Run

Instructions for running the project can be found in the README files located in the frontend and backend folders.

Alumni Portal Backend

How to Run

Using Python

  1. Open a shell/command line in the backend folder (preferably after activating a Python virtual environment).
  2. Install the required Python packages:

pip install -r requirements.txt

  1. Run the Python script:

python3 main.py

Using Docker

Here, the image tag name is alumni-backend; you can name it as you like. Also, the default port in Docker is 8000, and locally, you can map it to any port as you like.

  • Build the Docker image:

docker build -t alumni-backend

  • Run the Docker container, mapping port 8000 to the desired port:

docker run -p 8000:8000 -d alumni-backend

Inno-alumni-portal Frontend

How to Run Locally

  1. Open a shell/command line in the frontend folder.
  2. Install all required packages using npm:

npm install

  1. To build the project for production:

npm run build

  1. To start the project in development mode:

npm start

Technology Details

  • Python FastAPI: FastAPI is a modern, fast (high-performance), web framework for building APIs with Python.
  • Next.js: Next.js is a React framework that enables functionality like server-side rendering and generating static websites for React-based web applications.
  • Docker: Docker is a platform for developing, shipping, and running applications in containers.
  • PostgreSQL: PostgreSQL is an open-source relational database management system known for its reliability and robustness.

Additional Information

  • The backend (located in the 'app' folder) is developed using Python.
  • The frontend (located in the 'src' folder) is developed using Next.js, JavaScript, HTML, and CSS.
